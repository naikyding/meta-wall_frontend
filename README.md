# meta-wall_frontend [🔗](http://meta-wall.vercel.app/)

## 說明
MetaWall 是一個社群平台，使用者可以在這建立自已空間，進而發文與喜歡其它用戶的貼文。
## 開發技術
| 前端 | 後端 |
| --- | --- |
| Vue3 | NodeJS (Express) |
|Vite| MongoDB (Mongoose) |
| Pinia | JWT Auth |
| Tailwind CSS | -- |
| Remote 容器開發 | -- |

## 特點
- google、facebook 第三方帳號登入 
- 串接 imgur 圖床 (貼文圖片上傳)
- 發信驗證 (忘記密碼)
